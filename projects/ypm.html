Unique password managing

A simple, secure way of never having to remember a password again. Unlike other password managers, there's no master password for fraudsters to get their hands on. The only way to access Yoti Password Manager is with the Yoti app on your phone.

Cleaning up!

While the project itself is great and it's working properly, the quality of code needs improvement. Written 2 years ago by Java developers (they just learned script probably) is written in a quite unique way. That means high risk while refactoring, tears and anger, but eventually application is growing bigger and becoming better! We're trying to decouple classes, remove inheritance and favour composition. We've also switched to Preact to make extension faster and smaller.
